# 0x12-javascript-warm_up

This repository contains a series of JavaScript scripts aimed at building fundamental skills in JavaScript programming. Each script addresses specific tasks to reinforce key concepts.

## Scripts Overview

### 0. First constant, first print

- **File:** 0-javascript_is_amazing.js
- **Task:** Write a script that prints "JavaScript is amazing." 
- **Requirements:**
  - Create a constant variable called `myVar` with the value "JavaScript is amazing."
  - Use `console.log(...)` to print all output.
  - Do not use `var`.

### 1. 3 languages

- **File:** 1-multi_languages.js
- **Task:** Write a script that prints three lines with different programming language statements.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.

### 2. Arguments

- **File:** 2-arguments.js
- **Task:** Write a script that prints a message based on the number of arguments passed.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Use `process.argv` for reference.
  - Do not use `var`.

### 3. Value of my argument

- **File:** 3-value_argument.js
- **Task:** Write a script that prints the first argument passed to it.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Do not use `length`.

### 4. Create a sentence

- **File:** 4-concat.js
- **Task:** Write a script that prints two arguments passed to it in a specific format.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.

### 5. An Integer

- **File:** 5-to_integer.js
- **Task:** Write a script that prints "My number: \<first argument converted to integer\>" if the first argument can be converted to an integer.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Do not use `try/catch`.

### 6. Loop to languages

- **File:** 6-multi_languages_loop.js
- **Task:** Write a script that prints three lines using an array of strings and a loop.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Do not use any if/else statement.
  - Use only one `console.log`.
  - Use a loop (while, for, etc.).

### 7. I love C

- **File:** 7-multi_c.js
- **Task:** Write a script that prints "C is fun" x times.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Use only two `console.log`.
  - Use a loop (while, for, etc.).
  - Handle cases where the first argument can't be converted to an integer.

### 8. Square

- **File:** 8-square.js
- **Task:** Write a script that prints a square using the character X.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Use a loop (while, for, etc.).
  - Handle cases where the first argument can't be converted to an integer.

### 9. Add

- **File:** 9-add.js
- **Task:** Write a script that prints the addition of two integers.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Define a function with the prototype `function add(a, b)`.

### 10. Factorial

- **File:** 10-factorial.js
- **Task:** Write a script that computes and prints a factorial.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.
  - Do it recursively.
  - Use a function.

### 11. Second biggest!

- **File:** 11-second_biggest.js
- **Task:** Write a script that searches the second biggest integer in the list of arguments.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.

### 12. Object

- **File:** 12-object.js
- **Task:** Update the script to replace the value 12 with 89.
- **Requirements:**
  - Use `console.log(...)` to print all output.
  - Do not use `var`.

### 13. Add file

- **File:** 13-add.js
- **Task:** Write a function that returns the addition of two integers.
- **Requirements:**
  - The function must be visible from outside.
  - The name of the function must be `add`.
  - Do not use `var`.

## How to Use

1. Clone the repository:

```bash
git clone https://github.com/your-username/alx-higher_level_programming.git
```

2. Navigate to the JavaScript warm-up directory:

```bash
cd alx-higher_level_programming/0x12-javascript-warm_up
```

3. Run individual scripts:

```bash
./0-javascript_is_amazing.js
```

Make sure to replace `./0-javascript_is_amazing.js` with the desired script file.

Feel free to explore and enhance your JavaScript skills with these warm-up exercises!
